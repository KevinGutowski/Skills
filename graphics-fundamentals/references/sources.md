# Sources

This source map records the provenance for the graphics-fundamentals hub.

## Primary source

- Dan Hollick, *Making Software*, makingsoftware.com. The local references distill purchased chapters; quote sparingly and attribute chapter URLs.

## Local references

- `pixels-and-color.md` covers screens, color spaces/models/gamuts, contrast, blending, and digital images.
- `input.md` covers touch screens and input sensing.
- `fonts-and-vectors.md` covers curves, fonts, rasterization, SVG, and boolean operations.
- `effects-and-compression.md` covers blurs/noise/effects and image compression.

## Promotion rule

This skill is a mechanism backstop, not a design authority. Promote guidance only when it explains the underlying technical mechanism and routes practical decisions back to the owning task skill.
