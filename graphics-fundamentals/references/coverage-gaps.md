# Coverage Gaps

Use this file for technical graphics claims that need current standards, tooling, or implementation evidence.

## Candidate gaps

- **Standards drift:** WCAG/APCA status, browser color management, HDR/P3 behavior, image formats, font technologies, and SVG filter support need current verification before production advice.
- **Tool-specific behavior:** Figma, Photoshop, browser engines, OS renderers, and camera pipelines differ; verify exact behavior before prescribing tool steps.
- **Legal/accessibility boundaries:** Mechanism explanations do not replace accessibility compliance guidance; route compliance to the owning skill and current standards.
- **Performance fixtures:** Add examples for expensive blurs, SVG filters, image formats, and color conversions only with reproducible test cases.
- **Quote limits:** Do not expand local notes with long copied passages from purchased content.
