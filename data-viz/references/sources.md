# Data Visualization Sources

Use this as provenance for `data-viz`. It is not a quote bank.

## Chart Selection

- IBM Design Language charts guide — chart families by goal: comparison, trend, part-to-whole, correlation, relationships/connections, and maps.
- Atlassian essential chart types guide — practical chart-type rules for bars, lines, dot plots, tables, and exact values.
- Local distilled reference: `chart-selection.md`; decision matrix: `chart-selection/chart-decision-matrix.md`.

## Critique And Integrity

- Edward Tufte, *The Visual Display of Quantitative Information* — graphical excellence, graphical integrity, data-ink ratio, chartjunk, small multiples, and data density.
- Edward Tufte, *Envisioning Information* / analytical-design material — sparklines, layering and separation, micro/macro reading, escaping flatland, confections, and cause/effect evidence.
- Local distilled reference: `tufte.md`; detailed notes: `tufte/tufte-principles.md` and `tufte/analytical-design.md`.

## Editorial And Interactive Craft

- Hyperagent public skill `nyt-data-viz` from `alexmcdonnell-airtable/hyperagent-public-skills` — practitioner source for NYT/Upshot-style production patterns and D3 failure cases. Treat as source material, not as scripts or house style to import wholesale.
- Local distilled reference: `editorial-interaction.md`.

## Routing Notes

- Choose chart type first with `chart-selection.md`.
- Refine integrity, density, and visual encoding with `tufte.md`.
- Use `editorial-interaction.md` when the chart ships as a polished page, dashboard, or interactive artifact.
- Use `apple-design` (chart-experience-design) for the in-app experience around charts on Apple platforms, including Swift Charts and accessibility labels.
