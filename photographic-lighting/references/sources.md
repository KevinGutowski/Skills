# Sources

- Fil Hunter, Steven Biver, and Paul Fuqua, *Light: Science & Magic: An Introduction to Photographic Lighting*. Local Apple Books PDF extracted to `/tmp/books/light-science-magic.txt`. Primary source for hard/soft light, reflection families, metal, glass, portrait setup, white-on-white/black-on-black, and location lighting.
- Tony Northrup, *Tony Northrup's DSLR Book: How to Create Stunning Digital Photography*. Local Apple Books EPUB extracted to `/tmp/books/tony-northrup-dslr.txt`. Corroborating source for natural light, flash, bounce, fill, portraits, catchlights, and field practice.

Extraction note: `Light: Science & Magic` text extraction contains OCR/PDF-ligature drift (`Te`, `sofer`, `refection`). Quote exact text only after running `python3 scripts/verify_quotes.py`; prefer paraphrase when the extracted passage is damaged.
