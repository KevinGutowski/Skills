# Sources

This skill is based on one primary practitioner method with a corroborating checklist.

## Primary method

- Allan Peters, *Logos that Last: How to Create Iconic Visual Branding* (Rockport, 2023). Use for the brand noun process, sketch quantity, noun-combination methods, presentation protocol, badge design, brand evolution, and worked cases.

## Corroborator

- Gal Shir, *60 Tips for Logo Design* (2025; OCR'd image PDF, distill-only). Use only where it sharpens Peters' method: intake lanes, time-boxing, presentation, small-size checks, and application tests.

## Local references

- `worked-cases.md` holds Peters-method client cases and examples.

## Promotion rule

Promote a logo-design rule only when it is part of Peters' method, corroborated by Shir, or drawn from a reviewed logo project with the client context and outcome recorded.
