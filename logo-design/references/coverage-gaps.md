# Coverage Gaps

Use this file for logo-design candidates that need stronger project evidence before becoming reusable.

## Candidate gaps

- **AI originality checks:** Image search and generation workflows have changed since the sources; verify current search/generation tools before prescribing uniqueness checks.
- **Client presentation exemplars:** Add anonymized decks or before/after cases only when the subject-matter process, feedback, and outcome are recorded.
- **Industry-specific logo constraints:** Apparel, packaging, software, signage, and physical fabrication need examples before adding specialized rules.
- **Accessibility and trademark:** This skill can flag contrast/legibility/search risks, but legal trademark clearance and accessibility compliance need current expert/process checks.
- **Boundary with app icons:** Keep OS-contained app icon work in `apple-design`; do not promote app-icon rules into logo-system guidance.
