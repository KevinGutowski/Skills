# Coverage Gaps

Use this file for research methods, scripts, and thresholds that need stronger evidence or project context before becoming reusable guidance.

## Candidate gaps

- **Modern recruiting channels/tools:** The method pipeline is durable, but named tools and social channels are stale. Verify current options before recommending one.
- **AI-mediated research:** Add guidance for AI note-taking, synthesis, simulators, or interview agents only after ethics, consent, and accuracy risks are sourced.
- **Quantitative thresholds:** A/B, tree-test, lostness, and survey thresholds need project traffic, baseline, and statistical context before becoming rules.
- **Accessibility recruiting examples:** Add screeners and participant criteria only when they preserve both target-audience fit and assistive-technology expertise.
- **Research-to-skill promotion:** When research findings become agent guidance, include source, confidence, scope, and whether evidence is behavioral, attitudinal, or inferred.
