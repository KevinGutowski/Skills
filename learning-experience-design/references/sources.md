# Sources

This skill is intentionally compact and source-led.

## Primary method

- Julie Dirksen, *Design For How People Learn* (New Riders), supplies the learning-experience method: gap diagnosis, learner context, practice, feedback, motivation, environment, job aids, and transfer.

## Supplemental source

- Lidwell, Holden, and Butler, *Universal Principles of Design* (2003), supplies the advance-organizer pattern.

## Promotion rule

Promote learning guidance only when it changes behavior, not just content coverage. If a problem is environmental, motivational, or managerial, do not disguise it as a course.
