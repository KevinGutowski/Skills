# Coverage Gaps

Use this file for people-management guidance that needs legal, HR, or organization-specific review before becoming reusable.

## Candidate gaps

- **Legal/HR boundaries:** Hiring, firing, performance management, accommodations, compensation, discrimination, and leave policies need current HR/legal process checks.
- **Remote/hybrid norms:** Meeting, comms, and performance expectations have changed quickly; promote org rules only when the user's org context is known.
- **Manager training artifacts:** Add 1:1 templates, user guides, feedback scripts, or career packets only when accepted examples are available.
- **AI management tools:** Do not promote AI-generated review, feedback, or hiring guidance without privacy, bias, and policy checks.
- **Boundary with `design-org-influence`:** Influence up/sideways and the user's own promotion belongs there; this skill owns direct-report responsibility.
